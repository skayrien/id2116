Clean. Simple. Functional.



The Awakener is a simple, unobtrusive device that aids in ensuring high levels of concentration and alertness.
‍
Drive away sleepiness with a subtle tune or jerk yourself awake if you fall into too deep a slumber; the choice is yours! Do not fret that it would be overtly sensitive either! There is a delay programmed into the system, and if you choose to tilt your head at an angle, simply press the button to reset the system!

MicroBit Progress

The troubles we faced for the MicroBit was finally resolved this week, but we took some time in getting there due to some problems.

We realised that we should need to allow the users to reset their angle of elevation, and also that we should have a delay.

However, it was tricky because the code seemed to run through the moment the criteria was fulfilled, and the full song would play if the head of the user just tipped to the side for a moment.

We circumvented this by layering the tiers by 0.5 second intervals, which would allow the state of the object return to the stable state if it were to go back to the right angle.
