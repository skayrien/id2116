Awakener

We wish to make a tool to help to wake you up if you fall asleep in class.

The factors that we have to consider are:

  • Angle
  • Acceleration
  • Time that it stays there

    • We intend to set the parameters such that if the input of these factors exceed a certain point, then the Awakener would play a musical note for a length of time, causing the user to wake up!
